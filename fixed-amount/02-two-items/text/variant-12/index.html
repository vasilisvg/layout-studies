Bilingual layout

Sometimes we publish the same text in two languages next to each other, for instance English and German. If we use Josef Müller-Brockmann's measure theory, where the ideal measure is based on width of words, these two texts will have a different width. According to this simple tool, an English ideal measure of ten words for the Georgia is about 23.5em wide and the German ideal measure is 29em. This page is meant as an example of what this would look like. I used Google Translate to translate this text from English to German. My sincere appologies for the terrible translation. Please feel free to send me an accurate translation!

This layout idea is based on the book Grid Systems by Josef Müller-Brockman where the English and German versions of the same text are published next to each other. In the book, both text are exactly the same width though.

Zweisprachige Layout

Manchmal veröffentlichen wir den gleichen Text in zwei Sprachen neben einander, zum Beispiel Englisch und Deutsch. Wenn wir Josef Müller-Brockmann die Maßnahme Theorie, wo die ideale Maßnahme auf Breite von Wörtern basiert verwenden, werden diese beiden Texte eine unterschiedliche Breite aufweisen. Nach diesem einfachen Tool, ist ein englischer ideale Maß für zehn Worte für die Georgia über 23.5em breit und die deutsche Maßnahme ist ideal 29em. Diese Seite wird als ein Beispiel dessen, was diese aussehen würde gemeint. Ich habe Google Translate, diesen Text aus dem Englischen ins Deutsche zu übersetzen. Meine aufrichtigen Entschuldigungen für die schreckliche Übersetzung. Bitte zögern Sie nicht mir eine genaue Übersetzung zu senden!

Dieses Layout Idee basiert auf dem Buch Raster Systeme von Josef Müller-Brockman, wo die englischen und deutschen Versionen des gleichen Textes nebeneinander sind veröffentlicht wurde. In dem Buch sind sowohl Text genau die gleiche Breite though.
