One on top of the other

This is probably the most simple way to display two items on a page. The one always on top of the other. No CSS is needed to position the elements, this is how things flow naturally. So simple, you might actually forget this is a valid option.

Single, flexible column

This is the most simple, readable version of a layout. It's a heading, with a single paragraph of text with minimal styling. The layout is fluid for smaller screens, but it has a maximum width. This works on all devices, on most browsers. The text is always easy to read.
